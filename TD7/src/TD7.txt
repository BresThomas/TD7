La classe Collections du package java.util contient des méthodes pour manipuler des collections d'objets en Java. Certaines des méthodes disponibles dans la classe Collections incluent:

    sort: Trie les éléments dans l'ordre naturel ou en utilisant un comparateur personnalisé.
    binarySearch: Recherche un élément dans une liste triée.
    reverse: Inverse l'ordre des éléments dans une liste.
    shuffle: Mélange aléatoirement les éléments d'une liste.
    max et min: Trouve l'élément maximum ou minimum dans une collection.
    addAll: Ajoute tous les éléments d'une collection à une autre collection.

D'autre part, la classe Arrays du package java.util contient des méthodes pour manipuler des tableaux en Java. Certaines des méthodes disponibles dans la classe Arrays incluent:

    sort: Trie les éléments dans l'ordre naturel ou en utilisant un comparateur personnalisé.
    binarySearch: Recherche un élément dans un tableau trié.
    copyOf: Crée une copie d'un tableau avec une longueur donnée.
    fill: Remplit un tableau avec une valeur donnée.
    equals: Vérifie si deux tableaux sont égaux.
